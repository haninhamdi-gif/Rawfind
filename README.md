# Rawfind 

## 📌 Description du projet

Rawfind est une application intelligente dédiée à la recherche
de matières premières et à l'identification des fournisseurs
les plus adaptés 

L'objectif principal du système est d'aider les entreprises
ou les utilisateurs à sélectionner les fournisseurs les plus
pertinents en fonction de différents paramètres,
tels que le prix, la disponibilité,
la qualité et les exigences techniques.

Le système analyse les données disponibles,
compare plusieurs fournisseurs
et propose les solutions les plus appropriées
afin d'améliorer la prise de décision.

---

## 🎯 Objectifs du projet

Les principaux objectifs de Rawfind sont :

- Rechercher des matières premières disponibles
- Identifier les fournisseurs correspondants
- Comparer plusieurs fournisseurs
- Sélectionner les fournisseurs les plus adaptés
- Faciliter la prise de décision
- Réduire le temps de recherche des fournisseurs

---

## 🏗 Structure du projet

Le projet Rawfind est organisé en plusieurs dossiers
afin de faciliter la gestion et la maintenance du système.

###  backend

Le dossier **backend** contient la logique principale
du système. Il assure le traitement des données
et l'application des critères de recherche.

Il constitue le cœur technique de l'application.

#### 📄 app.py

Le fichier **app.py** représente le point d'entrée
principal du backend. Il permet d'initialiser
le système et de lancer les différentes fonctions.

#### bi_engine.py

Le fichier **bi_engine.py** contient le moteur
principal de traitement des données.

Il permet :

- d'analyser les données des matières premières
- d'appliquer les critères de recherche
- de comparer les fournisseurs
- de générer les résultats adaptés

Ce fichier représente l'élément central du système.

#### streamlit_app.py

Le fichier **streamlit_app.py** contient
l'interface utilisateur développée avec Streamlit.

Il permet à l'utilisateur :

- de sélectionner les critères
- de lancer la recherche
- d'afficher les résultats
- de comparer les fournisseurs

---

## Fonctionnement du système

Le fonctionnement du système Rawfind
se déroule selon les étapes suivantes :

1. L'utilisateur sélectionne les critères
   de recherche (matière première,
   caractéristiques, exigences, etc.)

2. Le backend analyse les données disponibles.

3. Le système compare les fournisseurs
   selon les critères choisis.

4. Les résultats les plus adaptés
   sont affichés à l'utilisateur.
